План работ по проекту "Распознавание лиц"

1) Теоретическая часть
 - введение
 - практическая ценность
 - описание работы видеокамеры
 - описание / назначение нейросетей
 - распознавание лиц с помощью нейросетей
 - взаимодействие со сторонними сервисами IOT-устройств
 - Arduino: описание, назначение
 - управление электронным замком через Arduino
 - структурная схема системы

2) Практическая часть
 - реализация модуля захвата изображений с видеокамеры
    - разработать модуль для захвата изображений с камеры
    - разработать модуль для захвата изображений с виртуальной камеры
 - реализация модуля распознавания лиц с помощью нейросетей
 - реализация модуля "Шина данных"
 - реализация модуля открывания двери
 - сборка устройства Arduino и написание программы
 - реализация модуля управления освещением
 - реализация модуля сбора статистики по работе системы
 - реализация клиентской программы для обучения системы
 - реализация программы для просмотра работы системы в реальной времени и статистики
